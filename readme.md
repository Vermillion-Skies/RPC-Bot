# INSTALLING  
## Linux  
1. In the releases tab, download the newest installerlinux.sh  
2. Make the file executable  
- Right click the file in a file manager and change the properties  
- In the terminal, enter the directory you downloaded the script to and type chmod +x installerlinux.sh  
3. Run the file in a terminal using ./installerlinux.sh

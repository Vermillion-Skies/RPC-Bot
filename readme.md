# RPC-Bot  
## By Vermillion-Skies  
### An application to interact with the Discord Rich Presence API in a fairly user-friendly format  
# INSTALLING  
## Linux  
1. In the releases tab, download the newest installerlinux.sh  
2. Make the file executable  
- Right click the file in a file manager and change the properties  
- In the terminal, enter the directory you downloaded the script to and type chmod +x installerlinux.sh  
3. Run the file in a terminal using ./installerlinux.sh  
- Make sure to do this in the directory you wish to install it to
## Windows  
1. In the releases tab, download the newest installwin.bat
2. Run the file, either from File Explorer or cmd.exe

# Setting up
1. Make a new app on the Discord Developer Portal  
2. Go to your app and copy the application ID  
3. In the RPCBot/assets, create a new file called pbcred.txt  
4. Paste your application ID in the first line and save the file  
# Running  
### Note: on your first run it WILL crash, as it doesn't have the config file made yet. This is an intentional behavior.
## Linux  
1. Enter the RPCBot folder  
2. Run startbot.sh  
## Windows
1. Enter the RPCBot folder
2. Run startbot.bat
